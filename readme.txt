The dataset was produced by the VDAIL lab at Xi'an University of Posts and Telecommunications.
The purpose of our dataset production is to provide experimental data for the processing method of video coloring for blue light laser knife surgery.
Our dataset contains three datasets R90, R140 and Animdata. 
Each dataset has three files Train, Gray and GT. 
The Train file contains 180 images used for training. 
The Gray file contains 60 grayscale images used for testing. 
The GT file contains the real images of the corresponding grayscale images in the Gray file.
